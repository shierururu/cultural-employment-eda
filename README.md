# cultural-employment-eda
Курсовая работа посвящена исследованию показателей занятости в сфере культуры на основе данных UNESCO за 2010-2016 годы. В теоретической части работы затрагиваются понятия культурного наследия, нематериального культурного наследия и скандинавской модели, а также рассматривается контекст культурных и креативных индустрий и оранжевой экономики.

В ходе описательного анализа и обработки данных применялся сервис Datawrapper. Итоговым результатом стал интерактивный дашборд, разработанный в Yandex DataLens, позволящий оценивать показатели с учетом гендерного фактора и проводить анализ различных сфер культуры.

## Используемые данные

| Файл | Описание |
| :--- | :--- |
| `data1.csv` | Доля лиц, занятых в сфере культуры и искусства |
| `data2.csv` | Доля женщин среди лиц, занятых в сфере культуры и искусства |
| `data3.csv` | Доля лиц, занятых в сфере культуры и искусства, которые работают в определенной сфере |
| `data4.csv` | Доля женщин среди лиц, занятых в определенной сфере культуры |
| `footnotes1.csv` | Заметки по подсчету данных |
| `footnotes2.csv` | Заметки по подсчету данных |
| `footnotes3.csv` | Заметки по подсчету данных |
| `footnotes4.csv` | Заметки по подсчету данных |

Работа выполнена в файле `data_work.ipynb`.
